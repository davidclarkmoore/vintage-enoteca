Class Distribution

A distribution is an area of service coverage. It consists of one or more polygon groups,a fill and stroke style for the groups, and a set of locations.
Defined in: service-areas.js.

{Number} distanceToLatLng(address, The)
Gives the distance from a google.maps.LatLng to the closest polygon in the distribution.A match is only successful if the given address matches the pattern defined forthe polygon group being checked. For example, this prevents a location in northernWashington state from indicating service is provided due to the proximity to theCanadian service area.
