<static> Locator.events.filterCheckboxClick(e)
Click event for the filter checkboxes. If the checkbox is being unchecked, it removes thelocations of that type from the map. If the checkbox is being checked, the locations of thattype will be added to the map. If zoomed out far enough, it also shows and hides statesdepending on the visible locations within that state (a state with no visible locations isnot displayed on the map).
Parameters:
e
Event object.
<static> Locator.events.infoWindowOpened()
Event fired when any info window is opened on the map. It generates the HTML for the smalland large sizes of the info window the first time that window is opened. It also maximizesthe info window if need be. Note that this event is attached to the map, not the markers,because the markers lose the event due to removeOverlay from the MarkerClusterer.
<static> Locator.events.mapMoveEnd(force)
Event fired when the movement of the map ends (including zooming). Updatesthe sidebar with a list of locations currently visible on the map, includinglocations that are within clusters.
Parameters:
{Boolean} force Optional, Default: false
Rebuild the sidebar regardless.
<static> Locator.events.mapZoom(oldLevel, newLevel)
Event fired when the map changes its zoom level. Hides or shows the states andaddress marker depending on the map configuration.
Parameters:
{Number} oldLevel
The previous zoom level of the map.
{Number} newLevel
The new zoom level of the map.
<static> Locator.events.markerClick(latlng, maximize)
Event fired when a marker is clicked. Sets up some variables used by Locator.events.infoWindowOpened.
Parameters:
{GLatLng} latlng
The GLatLng of the clicked marker. Not used.
{Boolean} maximize Optional, Default: false
True to maximize the info window for this marker. False to open the small info window.
<static> Locator.events.moreInfoClick(e)
Click event for the "More Info" links. Opens the large info window for the location.This event is also used by the location li when it is displayed in the sidebar.

e
Event object.
<static> Locator.events.multipleMatchesLinkClick()
Click event for an address link, which is displayed when a search is performed andthe search matches multiple locations. This updates the address and submits the searchagain.
<static> Locator.events.pagerPageLinkClick(e, page)
Click event for the pager links. Handles both the arrows and numbers. Generates thelocation sidebar HTML and inserts it if doesn't exist or displays the locations ifthey've already been generated.
Parameters:
e
Event object.
{Number} page Optional
The page to display. If omitted, the function tries to obtainthe page from the this object of the function, which should be an anchor tag.
